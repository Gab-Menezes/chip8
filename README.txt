Este projeto é um emulador de chip8 utilizando SDL2 (https://www.libsdl.org/download-2.0.php)
Para compilar é necessario incluir SDL2.lib e SDL2main.lib como dependencias adicionais
e copiar o arquivo SDL2.dll para o mesmo diretorio do executavel ou main.c.
Todos estes arquivos estão na pasta vendor
A compilação deve ser feita em x86.

Alem disso é necessario algumas ROM, algumas ja estao sendo enviadas com o programa.

Obrigado.
